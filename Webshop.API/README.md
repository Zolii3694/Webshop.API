A projekt egy ASP.NET Core Web API alapú webshop backend AngularJS frontendhez.
A rendszer termékeket és kategóriákat kezel, Entity Framework Core segítségével SQL adatbázisban tárolja az adatokat.
A vásárlók termékeket listázhatnak, kereshetnek, kategória szerint szűrhetnek és rendelést adhatnak le.
A rendelés során a rendszer menti a vásárlói adatokat, rendelési tételeket, valamint csökkenti a készletet.
Az alkalmazás JWT alapú bejelentkezést és admin jogosultságot használ.
Admin felhasználó termékeket és kategóriákat kezelhet, valamint megtekintheti a rendeléseket.
